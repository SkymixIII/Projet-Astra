1 Interfaces

interface Item

distance(autreItem)→ double
Précondition
autreItem != null ; this et autreItem possèdent des coordonnées initialisées.
Postcondition
Retourne un double ≥ 0 égal à la distance euclidienne entre les deux items. Aucun des deux items n'est modifié.
deplacer(x, y)→ void
Précondition
x ≥ 0, y ≥ 0 ; Carte.estDansLimites(x, y) == true ; la case cible est accessible.
Postcondition
this.x == x et this.y == y.
deplacer(direction)→ void
Précondition
direction != null ; la case adjacente dans cette direction existe et est accessible.
Postcondition
Les coordonnées de l'item sont incrémentées d'une unité dans le sens de direction.

interface Batiment extends Item

isOperationnel()→ boolean
Précondition
Aucune.
Postcondition
Retourne true ssi sante > 0. État du bâtiment non modifié.
affecterPersonnel(ouvrier)→ void
Précondition
ouvrier != null ; ouvrier.disponibilite == true ; personnelAffecte.size() < maxPersonnel.
Postcondition
personnelAffecte contient ouvrier ; ouvrier.disponibilite == false.
retirerPersonnel(ouvrier)→ void
Précondition
ouvrier != null ; ouvrier appartient à personnelAffecte.
Postcondition
ouvrier absent de personnelAffecte ; ouvrier.disponibilite == true.
subirDegats(quantite)→ void
Précondition
quantite > 0.
Postcondition
sante == max(0, sante_avant - quantite) ; si sante == 0, isOperationnel() retourne false.
reparer(quantite)→ void
Précondition
quantite > 0 ; sante < santeMax.
Postcondition
sante == min(santeMax, sante_avant + quantite).

interface EventListener

onEvent(evenement)→ void
Précondition
evenement != null.
Postcondition
L'événement est traité ; les effets dépendent du type concret de evenement et du composant abonné.
