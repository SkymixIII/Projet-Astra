1 Interfaces

interface Item

distance(autreItem: Item): double

AspectDétail
Signature complètedouble distance(Item autreItem)
PréconditionautreItem != null ; this.x, this.y, autreItem.x, autreItem.y sont initialisés (coordonnées valides).
PostconditionRetourne un double >= 0 égal à sqrt((autreItem.x - this.x)² + (autreItem.y - this.y)²). Ni this ni autreItem ne sont modifiés.

deplacer(x: int, y: int): void

AspectDétail
Signature complètevoid deplacer(int x, int y)
Préconditionx >= 0 ; y >= 0 ; Carte.estDansLimites(x, y) == true ; la case (x, y) est accessible (non bloquée).
Postconditionthis.x == x et this.y == y.

deplacer(direction: Direction): void

AspectDétail
Signature complètevoid deplacer(Direction direction)
Préconditiondirection != null ; la case adjacente dans direction existe dans la Carte et est accessible.
PostconditionLes coordonnées de l'item sont incrémentées d'une unité dans le sens de direction.
